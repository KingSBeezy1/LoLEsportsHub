# LoLEsportsHub

Repository for the **LoLEsportsHub** project — a web application built during my training/module to showcase core web development skills through a League of Legends–themed esports booking & information system.

---

## ⚙️ Technologies Used

### Backend
* **ASP.NET Core** – modern, high-performance web framework
* **Entity Framework Core** – ORM for database access
* **Newtonsoft.Json** – Advanced JSON serialization and deserialization.

### Frontend
* **Bootstrap 5** – responsive UI framework
* **Razor Views / MVC** – UI templating

### Testing
* **NUnit** – testing framework
* **Moq** – mocking library
* **MockQueryable** – simplifies LINQ mock setups

---

## 🚀 How to Run the Project

**1. Clone the repository:**

```bash
git clone https://github.com/<your-username>/LoLEsportsHub.git
```

**2. Navigate to the project directory:**

```bash
cd LoLEsportsHub
```

**3. Restore dependencies:**

```bash
dotnet restore
```

**4. Run the application:**

```bash
donet run
```

**5. Open your browser and navigate to:**

```bash
[git clone https://github.com/<your-username>/LoLEsportsHub.git](http://localhost:{localPort})
```
